<h2>__[소개글]__ </h2><br>
TQQQ/200MA를 기본으로 하는 전략들을 쉽게 모니터링 할 수 있도록
위젯으로 만든 앱입니다. <br>

앱에서 사용하는 권한은 인터넷 연결(네트워크 연결, 네트워크 상태)만 있습니다.<br>
앱 데이터 업데이트 주기는 1시간 단위 입니다 (자동 업데이트)<br>
새로고침(수동 업데이트)을 너무 자주 하지마세요. API 사용이 제한될 수 있습니다.<br>
